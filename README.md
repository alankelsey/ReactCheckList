# ReactCheckList
ReactCheckList for practice

built from https://handsonreact.com/

## Setup 
- clone repo
- `cd ReactCheckList`
- `npx create-react-app keeptrack --template typescript --use-npm`
- `cd keeptrack`
- `npm install mini.css@3.0.1`


## run
- `cd keeptrack`
- `npm start`
- access at http://localhost:3000
